# 📊 Customer Spending Analysis using MySQL

## 📌 Overview

This project analyzes customer spending data using MySQL.
It includes database creation, data cleaning, and business analysis using SQL queries.

---

## 🎯 Objectives

* Analyze customer spending behavior
* Identify top customers
* Calculate revenue metrics
* Understand sales trends

---

## 📊 Business Analysis Queries

This project contains **10 SQL queries** to answer key business questions:

1. Total number of customers
2. Total revenue generated
3. Average customer spending
4. Top 10 customers by spending
5. Monthly revenue analysis
6. Highest revenue month
7. Repeat customers identification
8. Low spending customers
9. Daily sales analysis
10. Customer purchase frequency

---

## 🚀 How to Run

1. Run `01_database_table.sql`
2. Import CSV dataset into MySQL
3. Run `02_data_cleaning.sql`
4. Run `03_analysis_queries.sql`

---

## 📈 Key Insights

* A small number of customers contribute most of the revenue
* Sales vary across different time periods
* Some customers make repeated purchases
* Spending patterns differ among customers
